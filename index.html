Does error catching seems hard for you? Did you ever see "Pyramid of Doom"?
Pyramid of Doom 
            
            step1(function (value1) {
                    step2(value1, function(value2) {
                        step3(value2, function(value3) {
                            step4(value3, function(value4) {
                                // Do something with value4
                            });
                        });
                    });
                });
            
        
            
            step1(function (err1, value1) {
                    if (err1) // Hande error
                    step2(value1, function(err2, value2) {
                        step3(value2, function(err3, value3) {
                            step4(value3, function(err4, value4) {
                                // Do something with value4
                            });
                        });
                    };
                });

Promisses

            
                Q.fcall(promisedStep1)
                .then(promisedStep2)
                ...
                .then(function (value4) {
                    // Do something with value4
                })
                .catch(function (error) {
                    // Handle any error from all above steps
                })
                .done();

they have three states

  • Fulfilled
  • Rejected
  • Pending
